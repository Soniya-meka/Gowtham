# Xenon International School - Futuristic Premium Website

This is a world-class, futuristic school website designed with a unique premium identity. It combines modern elegance with interactive UI elements to create a trust-building experience for parents and students.

## ✨ Features

- **Futuristic Hero Section**: Animated background, floating particles, and floating book animations.
- **Glassmorphism UI**: Transparent, blurred navigation and cards inspired by Apple and Tesla designs.
- **Interactive Elements**: Custom cursor glow, hover effects, and parallax scrolling.
- **Premium Typography**: Outfit and Inter fonts for a modern, luxurious feel.
- **Micro-interactions**: Smooth ScrollReveal animations and animated statistics counters.
- **Fully Responsive**: Mobile-first design that looks great on any device.
- **SEO Optimized**: Semantic HTML structure and descriptive metadata.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with advanced gradients, animations, and transitions.
- **JavaScript**: Smooth interactions and DOM manipulation.
- **Font Awesome**: Premium icons for a professional look.
- **ScrollReveal.js**: For elegant scroll-triggered animations.

## 📁 Directory Structure

- `index.html`: Main structure of the website.
- `style.css`: Core design system and animations.
- `script.js`: Interactive logic and animation triggers.
- `assets/`: 
  - `campus.png`: Ultra-modern campus hero background.
  - `principal.jpg`: Professional leadership portrait.
  - `gallery-*.jpg`: High-tech school life and facility images.
  - `event-*.jpg`: Academic and extracurricular event highlights.

## 🚀 How to Run

### Option 1: Using Local Dev Server (Recommended)
This approach provides hot-reloading and a better development experience.
1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Run `npm install` to install dependencies (Vite).
3. Run `npm start` or `npm run dev`.
4. Open the provided local URL (usually `http://localhost:5173`).

### Option 2: Direct Open
1. Simply open `index.html` in any modern web browser.
2. For the best experience, ensure you are connected to the internet to load Google Fonts and Font Awesome icons.
